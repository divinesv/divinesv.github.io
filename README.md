# DivineSV

DivineSV is a devotional website built with Next.js and designed for static hosting on GitHub Pages. It presents Hindu devotional content across deity pages, stotras, a small scripture library, and a Bhagavad Gita reading experience.

## Overview

- Next.js App Router
- TypeScript
- Custom CSS
- Static export for GitHub Pages

## Run Locally

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The production export is generated in `out/`.

## Deployment

This project is set up to publish through GitHub Pages from the repository:

`divinesv/divinesv.github.io`
